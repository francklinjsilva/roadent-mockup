📗 Integración Google Sheets

Conecta el CRM con tu hoja de cálculo en 3 pasos

1
Crea tu Google Sheet

Abre Google Sheets y crea un nuevo archivo llamado "RoaDent CRM". En la primera fila agrega estas columnas:

Fecha | Nombre | Contacto | Especialidad | Sede | Fecha Cita | Hora | Mensaje | Origen | Etapa | Notas
2
Crea el Script de Google Apps

Ve a Extensiones → Apps Script y pega este código:

function doPost(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var data = JSON.parse(e.postData.contents);
  sheet.appendRow([
    new Date(), data.nombre, data.contacto,
    data.especialidad, data.sede, data.fecha,
    data.hora, data.mensaje, data.origen,
    data.etapa, (data.notas||[]).join(' | ')
  ]);
  return ContentService.createTextOutput(JSON.stringify({ok:true}))
    .setMimeType(ContentService.MimeType.JSON);
}

Luego ve a Implementar → Nueva implementación → Aplicación web. Acceso: Cualquier persona. Copia la URL.

3
Pega la URL del Web App aquí

Una vez configurado, el botón "Sync Sheets" enviará todos los prospectos nuevos automáticamente.
